# Language Versions

English is the primary language in this repository.

## Core Documents

| Document | English | Chinese | French | German |
| --- | --- | --- | --- | --- |
| README | `README.md` | `README.zh-CN.md` | `README.fr.md` | `README.de.md` |
| Contributing | `CONTRIBUTING.md` | `CONTRIBUTING.zh-CN.md` | `CONTRIBUTING.fr.md` | `CONTRIBUTING.de.md` |
| Curated Projects | `PROJECTS.md` | `PROJECTS.zh-CN.md` | `PROJECTS.fr.md` | `PROJECTS.de.md` |
| Labels Guide | `.github/labels.md` | `.github/labels.zh-CN.md` | `.github/labels.fr.md` | `.github/labels.de.md` |
| PR Template Guide | `.github/PULL_REQUEST_TEMPLATE.md` | `.github/PULL_REQUEST_TEMPLATE.zh-CN.md` | `.github/PULL_REQUEST_TEMPLATE.fr.md` | `.github/PULL_REQUEST_TEMPLATE.de.md` |

## Notes

- Issue forms are available in English, Chinese, French, and German:
  - `01-idea.en.yml`, `11-idea.zh-CN.yml`, `21-idea.fr.yml`, `31-idea.de.yml`
  - `02-looking-for-solution.en.yml`, `12-looking-for-solution.zh-CN.yml`, `22-looking-for-solution.fr.yml`, `32-looking-for-solution.de.yml`
  - `03-building-update.en.yml`, `13-building-update.zh-CN.yml`, `23-building-update.fr.yml`, `33-building-update.de.yml`
- Issue forms are prefixed with numbers to keep display order stable in the template picker.
