# Contributing to Spark to Code

语言： [English](CONTRIBUTING.md) | **中文** | [Français](CONTRIBUTING.fr.md) | [Deutsch](CONTRIBUTING.de.md)

本仓库是创意协作社区，不是源码仓库。  
核心目标是把想法从痛点推进到实现。

## 参与渠道

1. 提交 Issue（默认）
- 提交新想法
- 寻找现成软件推荐
- 同步开发中项目进度

2. 在 Issue 下评论
- 补充具体使用场景
- 提供成熟产品官方链接
- 表达加入团队的意愿

3. 提交 PR（次要）
- 更新文档、模板、精选清单
- 优化仓库协作机制

## 提交 Issue 前

1. 先搜索已有 Issue，避免重复。
2. 使用正确的 Issue 表单。
3. 明确描述：谁在什么场景下遇到什么问题。

## 协作礼仪

1. 反馈应具体、可执行。
2. 推荐工具时请附官网或 GitHub 链接。
3. 想加入项目请写明：
- 角色
- 技术栈
- 每周可投入时间

## 创意公开共享约定

发布在本仓库的创意默认视为公开共享。  
请勿发布商业机密或敏感信息。

## 维护者建议流程

1. 每周进行一次 Triage：
- 打标签
- 去重合并
- 对已解决议题附链接后关闭

2. 对活跃开发议题：
- 引导团队在同一 Issue 持续更新里程碑与进度

3. 对信息不足议题：
- 标记 `needs-info`
- 长期无补充则关闭
